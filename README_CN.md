## 💡 为什么做这个项目？

几周前，我在 Reddit 上看到一个大学生发的帖子。他正在为毕业论文做一个 AI 辅助学习工具，但买不起 ChatGPT Plus，也没有信用卡去申请 API 额度。他找到的所有"免费 API 列表"要么已经过期，要么需要绑卡，要么有地区限制。

那个帖子让我一直放不下。

我自己的工作和产品大量使用 LLM API，有不少闲余的算力。与其浪费掉，不如分享出来。学生、独立开发者、爱好者——任何想体验 AI 的人，都不应该被一堵付费墙挡在门外。

**于是有了这个项目。** 这里的每一个 Key 都是真实可用的 API token。不需要信用卡，不需要注册，复制粘贴就能用。

如果这能帮到哪怕一个人完成他们的项目或论文，就值了。

## 🚀 如何使用

所有 Key 兼容 OpenAI API 格式，Base URL 为 `https://aiapiv2.pekpik.com/v1`。

### cURL

```bash
curl https://aiapiv2.pekpik.com/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer 从上方表格复制Key" \
  -d '{
    "model": "gpt-5.4",
    "messages": [{"role": "user", "content": "Hello!"}]
  }'
```

### Python (OpenAI SDK)

```python
from openai import OpenAI

client = OpenAI(
    base_url="https://aiapiv2.pekpik.com/v1",
    api_key="从上方表格复制Key"
)

response = client.chat.completions.create(
    model="gpt-5.4",
    messages=[{"role": "user", "content": "Hello!"}]
)
print(response.choices[0].message.content)
```

### Node.js (OpenAI SDK)

```javascript
import OpenAI from 'openai';

const client = new OpenAI({
  baseURL: 'https://aiapiv2.pekpik.com/v1',
  apiKey: '从上方表格复制Key',
});

const response = await client.chat.completions.create({
  model: 'gpt-5.4',
  messages: [{ role: 'user', content: 'Hello!' }],
});
console.log(response.choices[0].message.content);
```

### 搭配第三方工具

| 工具 | 设置方式 |
|------|---------|
| **Cursor** | Settings → Models → API Key 填入 Key，Base URL 填 `https://aiapiv2.pekpik.com/v1` |
| **ChatBox** | 设置 → API → 自定义提供商 → 填入 Base URL 和 Key |
| **LobeChat** | 设置 → 语言模型 → OpenAI → 填入 API Key 和 Proxy URL |
| **NextChat** | Settings → OpenAI → API Key 和 Base URL |
| **Open WebUI** | Admin → Settings → Connections → OpenAI API |
| **沉浸式翻译** | 设置 → 翻译服务 → OpenAI → 填入 Key 和自定义 API 地址 |
| **Bob 翻译** | 偏好设置 → 服务 → 翻译 → OpenAI → 填入 Key 和 API URL |

## ❓ 常见问题

### 这些 Key 是什么？

本项目提供的 Key（形如 `sk-xxx`）是我们自有平台签发的 API token，兼容 OpenAI API 格式。你可以用它调用 GPT-5.4、Claude Opus 4.6、DeepSeek、Gemini 等 90+ 模型。

### 和直接用 OpenAI 有什么区别？

| | 本项目 | 直接用 OpenAI |
|---|---|---|
| 需要信用卡 | ❌ 不需要 | ✅ 需要 |
| 免费额度 | ✅ 有 | ❌ 无 |
| 多模型切换 | ✅ 一个 Key 用所有模型 | ❌ 每家单独注册 |
| 中国大陆可用 | ✅ 直连 | ⚠️ 需要代理 |
| 最新模型 | ✅ GPT-5.4 / Claude 4.6 / Grok 4.20 | ✅ 仅 OpenAI |

### Key 有什么限制？

- 每个 Key 有预算上限（$3-$10 不等）
- 每个 Key 有速率限制（3-20 RPM）
- 每个 Key 有效期通常为 24-48 小时
- 超出限制返回 429 错误

### 额度用完 / Key 过期了怎么办？

不需要做任何事。本项目**每天自动更新 3-5 次**，过期 Key 会被自动清理，新 Key 会自动发布。

1. ⭐ **Star 本项目** — 让更多人发现
2. 👁️ **Watch → Releases** — 获取更新通知
3. 💬 **加入 Discussions** — 获取活动信息

### 安全吗？

- 这些是我们自有平台签发的 token，不是第三方厂商的 API Key
- 每个 Key 有独立的预算和速率限制
- 我们不记录对话内容
- Key 过期后自动失效

### 支持哪些地区？

全球可用，中国大陆可直连，无需代理。
