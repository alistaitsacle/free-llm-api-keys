## 🚀 如何使用

所有 Key 兼容 **OpenAI API 格式**，设置 Base URL 并粘贴 Key 即可。

**Base URL：** `https://aiapiv2.pekpik.com/v1`

### cURL

```bash
curl https://aiapiv2.pekpik.com/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer 粘贴Key" \
  -d '{
    "model": "gpt-5.4",
    "messages": [{"role": "user", "content": "你好！"}]
  }'
```

### Python

```python
from openai import OpenAI

client = OpenAI(
    base_url="https://aiapiv2.pekpik.com/v1",
    api_key="粘贴Key"
)

response = client.chat.completions.create(
    model="gpt-5.4",
    messages=[{"role": "user", "content": "你好！"}]
)
print(response.choices[0].message.content)
```

### Node.js

```javascript
import OpenAI from 'openai';

const client = new OpenAI({
  baseURL: 'https://aiapiv2.pekpik.com/v1',
  apiKey: '粘贴Key',
});

const response = await client.chat.completions.create({
  model: 'gpt-5.4',
  messages: [{ role: 'user', content: '你好！' }],
});
console.log(response.choices[0].message.content);
```

### 图像生成 (DALL-E 3)

```python
response = client.images.generate(
    model="dall-e-3",
    prompt="一只猫在太空中漂浮，数字艺术风格",
    size="1024x1024",
    n=1
)
print(response.data[0].url)
```

### 语音合成 (TTS)

```python
response = client.audio.speech.create(
    model="tts-1-hd",
    voice="alloy",
    input="你好！这是一个免费的语音合成示例。"
)
response.stream_to_file("output.mp3")
```

### 文本向量化 (Embeddings)

```python
response = client.embeddings.create(
    model="text-embedding-3-small",
    input="快速的棕色狐狸跳过了懒惰的狗"
)
print(f"向量维度: {len(response.data[0].embedding)}")
```

<summary><b>这些 Key 是什么？</b></summary>

平台自有签发的 API token（`sk-xxx`），兼容 OpenAI API 格式。一个 Key 可以访问 90+ 模型。

<summary><b>和直接用 OpenAI 有什么区别？</b></summary>

| | 本项目 | 直接用 OpenAI |
|---|---|---|

| 多模型 | ✅ 一个 Key 用所有模型 | ❌ 每家单独注册 |

| 最新模型 | ✅ GPT-5.4, Claude 4.6, Grok 4.20, Gemini 3.1 | ✅ 仅 OpenAI |
</details>

<details>
<summary><b>有什么限制？</b></summary>

- 每个 Key 预算 **$20 – $100**
- 速率限制：每分钟 3-20 次请求 (RPM)
- 有效期 24-48 小时
- Key 是公开共享的，额度可能被他人消耗
</details>

<details>
<summary><b>Key 不能用了？</b></summary>

Key 是公开共享的，额度可能已被用完。本项目**每天自动更新 3-5 次**——稍后回来即可。也可以 **Watch → Releases** 接收通知。
</details>

<details>
<summary><b>安全吗？</b></summary>

- 平台自有 token，不是第三方厂商的 Key
- 每个 Key 有独立预算和速率限制
- 不记录对话内容
- 24-48 小时后自动失效
</details>

<details>
<summary><b>支持哪些地区？</b></summary>

全球可用，中国大陆可直连，无需代理。
</details>
